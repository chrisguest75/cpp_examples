# README

Demonstrates how to use ghidra to dissassmble code  

## Install

```sh
# replaces adoptopenjdk
brew install --cask temurin

# install ghidra
brew install ghidra
```

## Decompile

```sh
ghidraRun
```

Steps:

* Create a project in this folder.
