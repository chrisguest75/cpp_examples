## Install

REF: [github.com/chrisguest75/python_examples/48_pipx/README.md](https://github.com/chrisguest75/python_examples/blob/main/48_pipx/README.md)

```sh
# select the version of python that has pipx
pyenv global 3.11.7

pipx install conan
conan
conan version
```

## Start

```sh
conan profile detect
just build
```
