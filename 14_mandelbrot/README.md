# README

Demonstrate loading writing out a PNG.  

Ref: `vscode` extensions [README.md](../README.md)  

## Build and Run

Select the Terminal > Run Build Task command.

```sh
# create make
just create

# build target
just build

# run 
just run --generate 

# cleanup
just clean
```

Use the debugger in `vscode`
